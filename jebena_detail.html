Project Description

Jebena Cafe is a modern, responsive website created for a local Ethiopian coffee shop. The site elegantly displays the cafe’s atmosphere, menu offerings, and community culture. It enables customers to place orders or make reservations using EmailJS-powered forms, ensuring the cafe stays connected with visitors. Fully optimized for mobile and desktop, the project captures the authentic warmth of a real cafe while maintaining high performance and modern UI trends aligned with Tesfab’s dark-themed project portfolio.

Technologies Used
