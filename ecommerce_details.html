  • Order tracking with detailed history for users

Live Demo

Project Description

This e-commerce platform is a fully functional shoe store built using HTML, CSS, JavaScript, PHP, and Firebase for authentication. It provides a seamless shopping experience with dynamic product browsing, wishlist and cart management, and secure Stripe-powered checkout. The admin dashboard enables easy management of products, orders, users, and blog content. Additional features include AJAX-based cart updates, order tracking, and a blogging system with likes and comments. The platform is designed with a mobile-first approach ensuring responsiveness and smooth user interaction across all devices.

Technologies Used

HTML CSS3 JavaScript PHP Firebase Stripe API MySQL AJAX
